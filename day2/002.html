1. 서버 : 사용자가 요청하면 서비스를 제공하는 프로그램이다 2. 웹서버(http 서버) : 사용자가 요청하면 html을 보내주는 서버 프로그램이다 (우리가 아는 모든 웹사이트는 http서버가 돌린다) 웹서버는 @기억하자@ 3. http는 웹에서 사용하는 통신표준 4. http요청을 보낼려면 서버의 ip주소와 포트를 알아야한다 5. 사용자는 ip를 기억하기 어렵다 그래서 www.naver.com과 같은 dimain name을 사용한더 6. DNS: 도메인 이름을 IP주소로 바꿔주는 서비스 사용자가 도메인 이름을 입력하는 DNS가 IP로 변환한다
